# Mini Player — 桌面迷你音乐播放器

基于 Python + PySide6 的 Windows 桌面迷你音乐播放器，半透明暗灰磨砂风格，400×50 迷你尺寸，始终置顶。

## 项目结构

```
mini_player/
├── main.py                  # 入口：创建 QApplication 和 MiniBar
├── requirements.txt         # PySide6, pygame, mutagen
├── resource/                # SVG 图标（浅灰白色 #cdcdd8）
│   ├── play.svg / pause.svg
│   ├── previous.svg / next.svg
│   ├── repeat_all.svg / repeat_single.svg / shuffle.svg
│   └── volume_high.svg / volume_medium.svg / mute.svg
├── core/
│   └── player.py            # 音乐播放引擎（pygame.mixer.music）
├── ui/
│   ├── mini_bar.py          # 主窗口 + 控制栏 + 事件过滤拖拽
│   └── acrylic.py           # Windows DWM 亚克力模糊（备用）
├── utils/
│   └── format.py            # 时间格式化 + 音频扩展名列表
└── dist/
    └── MiniPlayer/
        ├── MiniPlayer.exe   # 打包好的可执行文件（4.9 MB）
        └── _internal/       # 运行时依赖 + 图标资源
```

## 功能清单

| 功能 | 说明 |
|------|------|
| 播放/暂停 | 同一按钮切换，无歌曲时弹出文件对话框 |
| 上一首/下一首 | 切换播放列表中的曲目 |
| 循环模式 | 列表循环 → 单曲循环 → 随机播放，同一按钮循环切换 |
| 进度条 | 点击或拖拽跳转播放位置 |
| 音量控制 | 滑块调节 + 按钮静音（记忆静音前音量） |
| 音量图标 | 50-100% 显示高音量，1-49% 显示中音量，0% 显示静音 |
| 拖放 | 直接拖音频文件到窗口播放 |
| 右键菜单 | 打开文件 / 退出 |
| 窗口拖拽 | 从滑块以外的任意区域长按移动窗口 |

## 窗口特性

- **尺寸**: 400 × 50 px
- **背景**: 半透明暗灰色（rgba 24,24,32,140），`paintEvent` 绘制圆角矩形
- **置顶**: `WindowStaysOnTopHint`，始终在最前
- **无边框**: `FramelessWindowHint`，不显示标题栏和任务栏图标（`Tool`）
- **透明度**: `WA_TranslucentBackground` 实现背景透明，桌面壁纸可见

## 技术架构

```
main.py
  └── MiniBar (QWidget)          ← 窗口 + 控制栏
        ├── MusicPlayer (QObject) ← 播放引擎
        │     ├── pygame.mixer.music  音频解码播放
        │     ├── mutagen              元数据读取
        │     └── QTimer 100ms         进度轮询 + 曲目结束检测
        ├── QPushButton × 6    ← 图标按钮（SVG QIcon）
        ├── QSlider × 2        ← 进度条 + 音量滑块
        └── QLabel             ← 时间标签
```

### 播放引擎 (MusicPlayer)

- **状态**: `stopped` / `playing` / `paused`
- **播放列表**: 自动加载打开文件所在目录的所有音频文件
- **曲目结束检测**: QTimer 100ms 轮询 `pygame.mixer.music.get_busy()`
- **保护期**: 切歌后 500ms 内跳过结束检测，防止 `get_busy()` 短暂返回 False 导致误跳过
- **循环模式**:
  - `REPEAT_ALL` (0): 播完自动切下一首，列表尾回第一首
  - `REPEAT_SINGLE` (1): 播完重新播放当前曲目，进度重置到 0:00
  - `SHUFFLE` (2): 播完随机切到列表内任意一首曲目

### 窗口拖拽机制

通过 `QApplication.instance().installEventFilter(self)` 安装应用级事件过滤器：
- 从 MiniBar 及其子控件的鼠标事件中拦截左键操作
- 排除 `QSlider`（进度条和音量条保持原生行为）
- 移动超过 5px 阈值进入拖拽模式，消费事件阻止传递到子控件
- 短点击正常传递给按钮

### 图标系统

- 所有按钮使用 SVG 图标（`resource/*.svg`）
- 颜色统一为 `#cdcdd8`（浅灰白），通过 `sed` 批量替换 `fill="black"` / `stroke="black"`
- 通过 `QIcon(str(path))` 加载，`setIconSize(QSize(18, 18))` 固定尺寸

## 运行方式

**源码运行：**
```bash
pip install -r requirements.txt
python main.py
```

**EXE 运行：**
```
直接双击 dist/MiniPlayer/MiniPlayer.exe
```

## 打包命令

```bash
pyinstaller --windowed --name "MiniPlayer" \
    --add-data "resource;resource" \
    --hidden-import pygame --hidden-import mutagen \
    --clean main.py
```

输出在 `dist/MiniPlayer/`，整个文件夹复制到任意 Windows 电脑即可运行，无需安装 Python。
